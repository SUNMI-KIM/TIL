## 문제 풀이 방식


이 문제는 일반적인 최단 거리 BFS를 변형하여 진행하는 문제이다. 배열을 이동하는 과정에서 벽을 한 번 부술 수 있다는 조건이 있기 때문에, 벽을 부수거나 부수지 않은 상태를 따로 관리해야 한다. 

처음에는 상태 변화가 있으니 깊이 우선 탐색(DFS)을 활용한 백트래킹을 생각했으나, 배열의 범위가 최대 $1000 * 1000$이므로 시간 초과가 발생한다. 따라서 BFS로 풀이하면서 현재 경로가 벽을 깼는지 안 깼는지를 함께 기억하며 탐색해야 한다.

### 3차원 방문 배열 사용

그러면 방문 배열이 벽을 깬 최단 경로와 벽을 안 깬 최단 경로를 함께 관리해야 한다. 이는 3차원 방문 배열 (`visited[x][y][broken]`) 을 통해 이를 관리할 수 있다.`visited[x][y][0]`은 벽을 한 번도 부수지 않고 도달한 방문 기록이고, `visited[x][y][1]`은 벽을 한 번 부순 상태로 도달한 방문 기록이다.

탐색 과정에서 벽을 부순 후 예전에 지나왔던 빈칸으로 되돌아가는 경로가 큐에 담길 수 있지만, BFS의 특성 상 이런 비효율적인 경로는 최단 거리를 갱신하지 못하고 자연스럽게 도태되므로 정답에는 영향을 주지 않는다. 

### BFS 큐에 들어갈 클래스

`visited` 배열을 `boolean` 타입으로 설정했기 때문에 큐에 들어가는 좌표 정보는 자신의 현재까지 이동 거리와 벽 파괴 여부를 가지고 있어야 한다. 

```java
class Pair {
	int x;
	int y;
	int distance;
	int broken; // 0: 벽을 부수지 않음, 1: 벽을 부숨
	
	Pair(int x, int y, int distance, int broken) {
		this.x = x;
		this.y = y;
		this.distance = distance;
		this.broken = broken;
	}
}
```

### BFS 핵심 로직

```java
while (!queue.isEmpty()) {
			Pair pair = queue.poll();
			int x = pair.x;
			int y = pair.y;
			int distance = pair.distance;
			int broken = pair.broken;
			
			if (x == n - 1 && y == m - 1) return distance;
			
			for (int i = 0; i < 4; i++) {
				int nx = x + dx[i];
				int ny = y + dy[i];
				int nextDistance = distance + 1;
				int nextBroken = broken;
				
				if (nx < 0 || nx >= n || ny < 0 || ny >= m) continue;
				if (visited[nx][ny][broken]) continue;
				if (arr[nx][ny] == '1') {
					if (broken == 1) continue;
					nextBroken = 1;
				}
				visited[nx][ny][nextBroken] = true;
				queue.add(new Pair(nx, ny, nextDistance, nextBroken));
			}
		}
		return -1;
	}

```

탐색 중 다음 칸을 만났을 때 벽을 깬 상태인지 아닌지에 따라 다음과 같이 로직이 나뉜다. 

1. 다음 칸이 빈칸(0)인 경우 : 현재 상태(broken)를 그대로 유지하며 큐에 넣고 진행한다. 
2. 다음 칸이 벽(1)인 경우 : 
    - 내가 아직 벽을 깨지 않은 상태라면, 벽을 깨고 상태를 1로 변경한 뒤 큐에 넣는다.
    - 내가 이미 벽을 깬 상태라면, 더 이상 벽을 통과할 수 없으므로 탐색을 생략한다.

## 최종 코드 (Java)

```java
import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.util.ArrayDeque;
import java.util.Queue;
import java.util.StringTokenizer;

public class _2206 {
	
	static int[] dx = {1, 0, -1, 0};
	static int[] dy = {0, 1, 0, -1};
	
	static int n;
	static int m;
	static char[][] arr;

	public static void main(String[] args) throws Exception {
		
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = new StringTokenizer(br.readLine());
		
		n = Integer.parseInt(st.nextToken());
		m = Integer.parseInt(st.nextToken());
		arr = new char[n][];
		
		for (int i = 0; i < n; i++) {
			arr[i] = br.readLine().toCharArray();
		}
		
		System.out.println(solve());
		
	}
	
	static int solve() {
		int min = Integer.MAX_VALUE;
		
		Queue<Pair> queue = new ArrayDeque();
		boolean[][][] visited = new boolean[n][m][2];
		queue.add(new Pair(0, 0, 1, 0));
		visited[0][0][0] = true;
		
		while (!queue.isEmpty()) {
			Pair pair = queue.poll();
			int x = pair.x;
			int y = pair.y;
			int distance = pair.distance;
			int broken = pair.broken;
			
			if (x == n - 1 && y == m - 1) return distance;
			
			for (int i = 0; i < 4; i++) {
				int nx = x + dx[i];
				int ny = y + dy[i];
				int nextDistance = distance + 1;
				int nextBroken = broken;
				
				if (nx < 0 || nx >= n || ny < 0 || ny >= m) continue;
				if (visited[nx][ny][broken]) continue;
				if (arr[nx][ny] == '1') {
					if (broken == 1) continue;
					nextBroken = 1;
				}
				visited[nx][ny][nextBroken] = true;
				queue.add(new Pair(nx, ny, nextDistance, nextBroken));
			}
		}
		return -1;
	}

}

class Pair {
	int x;
	int y;
	int distance;
	int broken;
	
	Pair(int x, int y, int distance, int broken) {
		this.x = x;
		this.y = y;
		this.distance = distance;
		this.broken = broken;
	}
	
}
```
