## 문제 풀이 방식


예전에 이 문제를 풀었을 때는 톱니바퀴 배열 자체를 회전시켜야 한다고 생각했다. 입력 크기가 매우 작기 때문에, 실제로 배열을 직접 회전시키더라도 시간 제한 내에는 충분히 통과할 수 있다. 

또한 자바에서 `Deque`를 이용하면 회전을 효율적으로 처리할 수 있고, 자료구조를 잘 알고 있다면 비교적 직관적으로 해결할 수 있는 문제이기도 하다. 

하지만 `Deque`를 사용하지 않고, 인덱스를 이용해 회전을 표현하는 방식을 연습해보고 싶었기 때문에 각 톱니바퀴의 기준 인덱스를 관리하는 방식으로 구현했다.

---

### 톱니가 움직일 때, 배열을 직접 돌리는가?

배열을 물리적으로 회전시키는 방식도 가능하지만, 회전 연산 자체를 줄이고 싶었기 때문에 배열은 그대로 두고 인덱스만 이동시키는 방식을 선택했다. 

> Deque의 회전 연산 자체는 `O(n)` 이므로 자료구조를 잘 이용하도록 하자.
> 

즉, 각 톱니바퀴는 문자열 형태로 고정해 두고, 각 톱니의 12시 방향을 가르키는 인덱스만 관리한다. 

```java
int[] idx_arr = new int[4]; 
```

이렇게 하면 실제 회전을 인덱스 연산만으로 표현할 수 있다. 

---

### 회전을 동시에 처리해야 하는데, 어떻게 처리하지?

이 문제의 핵심은 회전이 연쇄적으로 전파되지만, 실제 회전은 동시에 발생한다는 점이다. 

만약 톱니를 검사하면서 바로 회전을 적용해 버리면, 이후 톱니들의 판단 기준이 바뀌어 잘못된 결과가 나온다. 

따라서 아래와 같은 과정으로 로직을 전개했다. 

1. 어떤 톱니가 어떤 방향으로 회전해야 하는지를 `rotate` 배열에 먼저 저장한다. 
2. 모든 판단이 끝난 뒤, 한 번에 회전을 적용했다. 

```java
public static void move_left(String[] str_arr, int[] idx_arr, int idx, int d, int[] rotate) {
		rotate[idx] = d;
		while (idx > 0) {
			if (!check(str_arr[idx - 1], str_arr[idx], (idx_arr[idx - 1] + 2) % 8 , (idx_arr[idx] - 2 + 8) % 8)) // 왼쪽과 오른쪽 잘 확인하기
				break;
			rotate[idx - 1] = rotate[idx] * -1;
			idx--;
		}
	}
```

### 회전 적용 (`spin`)

회전 방향이 결정된 후에는, 각 톱니의 12시 방향 인덱스만 변경해 주면 된다. 

- 시계 방향 → 인덱스 -1
- 반시계 방향 → 인덱스 +1
- 음수 방지를 위해 `(x + 8) % 8` 패턴 사용

```java
public static void spin(int[] rotate, int[] idx_arr) {
		for (int i = 0; i < 4; i++) {
			if (rotate[i] == -1) {
				idx_arr[i] = (idx_arr[i] + 1) % 8;
			} else if (rotate[i] == 1) {
				idx_arr[i] = (idx_arr[i] - 1 + 8) % 8; // 음수 인덱스는 이렇게 하는거야
			}
		}
	}
```

### 기준 톱니의 중복 회전 문제

처음 구현할 때는 spin() 함수를 `move_left()`와 move_right() 내부에서 각각 호출했다. 

하지만 이렇게 하면 기준 톱니가 두 번 회전하는 문제가 발생한다. 

이를 해결하기 위해 `move_left()`와 `move_right()`에서는 회전 방향만 기록한다. 모든 판단이 끝난 후 `spin()`을 단 한 번만 호출한다. 

---

### 인덱스 기준으로 3시/9시 방향 계산

각 톱니의 기준 인덱스는 12시 방향을 의미한다. 

- 3시 방향 : 기준 인덱스에서 `-2`

```java
(idx_arr[idx] - 2 + 8) % 8
```

- 9시 방향 : 기준 인덱스에서 `+2`

```java
(idx_arr[idx - 1] + 2) % 8
```

원형 구조이기 때문에 항상 `(값 + 8) % 8` 형태로 계산하여 인덱스 범위를 유지한다. 

## 최종 코드 (Java)

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.Arrays;
import java.util.StringTokenizer;
import java.util.stream.IntStream;

public class _14891 {

	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

		String[] str_arr = new String[4];
		int[] idx_arr = new int[4]; 
		for (int i = 0; i < 4; i++) {
			str_arr[i] = br.readLine();
		}

		int t = Integer.parseInt(br.readLine());
		int score = 0;
		for (int test_case = 0; test_case < t; test_case++) {
			StringTokenizer st = new StringTokenizer(br.readLine());
			int n = Integer.parseInt(st.nextToken()) - 1;
			int d = Integer.parseInt(st.nextToken());

			int[] rotate = new int[4]; 
			move_left(str_arr, idx_arr, n, d, rotate);
			move_right(str_arr, idx_arr, n, d, rotate);
			spin(rotate, idx_arr);
		}
		for (int i = 0; i < 4; i++) {
			if (str_arr[i].charAt(idx_arr[i]) == '0') continue;
			score += (int) Math.pow(2, i);
		}
		System.out.println(score);
	}

	public static void move_left(String[] str_arr, int[] idx_arr, int idx, int d, int[] rotate) {
		rotate[idx] = d;
		while (idx > 0) {
			if (!check(str_arr[idx - 1], str_arr[idx], (idx_arr[idx - 1] + 2) % 8 , (idx_arr[idx] - 2 + 8) % 8)) 
				break;
			rotate[idx - 1] = rotate[idx] * -1;
			idx--;
		}
	}

	public static void move_right(String[] str_arr, int[] idx_arr, int idx, int d, int[] rotate) {
		rotate[idx] = d;
		while (idx < 3) {
			if (!check(str_arr[idx], str_arr[idx + 1], (idx_arr[idx] + 2) % 8, ((idx_arr[idx + 1] - 2 + 8)) % 8)) 
				break;
			rotate[idx + 1] = rotate[idx] * -1;
			idx++;
		}
	}

	public static void spin(int[] rotate, int[] idx_arr) {
		for (int i = 0; i < 4; i++) {
			if (rotate[i] == -1) {
				idx_arr[i] = (idx_arr[i] + 1) % 8;
			} else if (rotate[i] == 1) {
				idx_arr[i] = (idx_arr[i] - 1 + 8) % 8; 
			}
		}
	}

	public static boolean check(String left, String right, int leftIdx, int rightIdx) {
		if (left.charAt(leftIdx) != right.charAt(rightIdx))
			return true;
		return false;
	}

}
```
