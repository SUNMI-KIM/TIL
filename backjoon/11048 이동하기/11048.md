## Dynamic Programming


DP(동적 계획법)는 하나의 큰 문제를 여러 개의 작은 부분 문제로 나누어 해결하고, 그 결과를 저장해 두었다가 재활용하여 전체 문제를 해결하는 알고리즘이다. 주로 점화식을 도출하여 문제를 푼다. 

DP를 사용할 때는 부분 문제들 사이의 점화식과 초기 조건을 잘 정의한 뒤 코드로 만들면 된다.

### DP 사용 조건

---

1. **Overlapping Subproblems ( 겹치는 부분 문제 )**

DP는 기본적으로 문제를 나누고 그 문제의 결과 값을 저장하여 재활용한다. 동일한 작은 문제들이 반복해서 계산되어야 하는 경우에 사용할 수 있다.

1. **Optimal Substructure ( 최적 부분 구조 )**

부분 문제의 최적 결과 값을 조합해 전체 문제의 최적 결과를 낼 수 있는 경우를 의미한다. 

→ 즉, 작은 문제들의 최적해를 바탕으로 큰 문제의 해를 도출할 수 있다면 DP를 사용할 수 있다. 

## 문제 풀이 방식


2차원 배열에서 DP를 해야 해서 처음에는 까다로울 것이라 생각했으나, 탐색 범위만 늘어났을 뿐 지금까지 풀어본 DP 중 점화식은 가장 직관적이었다.

문제에서 이동할 수 있는 방향은 오른쪽, 아래, 오른쪽 아래 대각선이다. 

이것을 목적지 `(x, y)` 입장에서 반대로 생각하면, `(x, y)`로 오기 직전의 위치는 왼쪽, 위, 왼쪽 위 대각선 3가지 중 하나가 된다. 

따라서 2차원 배열을 순회하면서, 현재 위치가 유효한 범위라면 현재 칸의 사탕 개수와 이전 3가지 방향에서 가져올 수 있었던 최대 사탕 개수를 더한 값을 비교하여 최댓값으로 현재 위치의 DP 배열을 갱신했다. 

## 최종 코드 (Java)

```java
import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.util.StringTokenizer;

public class _11048 {
	
	static int[] dx = {0, -1, -1};
	static int[] dy = {-1, 0, -1};
	
	static int n;
	static int m;
	static int[][] arr;

	public static void main(String[] args) throws Exception {
		
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = new StringTokenizer(br.readLine());
		
		n = Integer.parseInt(st.nextToken());
		m = Integer.parseInt(st.nextToken());
		arr = new int[n][m];
		
		for (int i = 0; i < n; i++) {
			st = new StringTokenizer(br.readLine());
			for (int r = 0; r < m; r++) {
				arr[i][r] = Integer.parseInt(st.nextToken());
			}
		}
		System.out.println(solve());
	}
	
	static int solve() {
		int[][] dp = new int[n][m];
		for (int x = 0; x < n; x++) {
			for (int y = 0; y < m; y++) {
				dp[x][y] = arr[x][y];
				for (int d = 0; d < 3; d++) {
					if (check(x + dx[d], y + dy[d])) continue;
					dp[x][y] = Math.max(dp[x][y], arr[x][y] + dp[x + dx[d]][y + dy[d]]);
				}
			} 
		}
		return dp[n - 1][m - 1];
	}
	
	static boolean check(int x, int y) {
		return 0 > x || n <= x || 0 > y || m <= y; 
	}

}

```
