## 트리의 지름


![](https://velog.velcdn.com/images/hariaus/post/98587a16-cdfb-4796-91e6-ce1f9e5af0aa/image.png)


트리의 지름이란, 트리 내에 있는 임의의 두 노드 사이의 경로 중 가장 긴 경로의 길이(가중치의 합)를 의미한다.

트리의 지름을 구할 때, 루트 노드를 거치지 않더라도 긴 경로가 존재할 수 있다. 또한 그래프 형태에서는 자식 노드의 제한이 없어, 왼쪽, 오른쪽을 나누어 탐색하기가 까다롭다. 

따라서 임의의 정점에서 탐색을 시작한 후, 가장 먼 노드(가중치의 합이 가장 큰 노드)를 탐색한다. 또한 찾아낸 가장 먼 노드에서 다시 한번 탐색을 진행한다.

### 1. 임의의 정점에서 탐색을 시작하여 가장 먼 노드를 찾는다.

탐색(DFS/BFS)를 통해 현재 정점(루트 노드)에서 가중치의 합이 가장 큰 노드를 찾는다. 위 그림에선 1 → 3 → 7 → 11 의 경로가 가장 가중치 합이 큰 경로이다. 

### 2. 찾은 가장 먼 노드에서 다시 한번 탐색을 진행한다.

1번 단계에서 찾은 지름의 한쪽 끝점에서 출발해, 다시 가장 가중치 합이 큰 경로를 찾는다. 이때 나온 최대 거리가 바로 트리의 지름이 된다. 위 그림에선 11 → 7 → 3 → 1 → 2 → 5 → 9 의 경로가 가장 가중치 합이 긴 경로이다.

## 문제 풀이 방식


위 로직을 코드로 구현하면 아래와 같다. 두 번의 DFS를 실행하기 위해, 최대 거리(`maxDistance`)와 그때의 노드 인덱스(`maxIndex`)를 전역 변수로 관리하며 갱신한다. 

```java
maxDistance = 0;
solve(new boolean[n + 1], 0, 1);
solve(new boolean[n + 1], 0, maxIndex);
```

DFS 메서드 내부에서는 가중치 합이 기존 maxDistance 보다 커질 때마다 값을 계속해서 갱신한다. 형태는 DFS와 동일하다. 

```java
static void solve(boolean[] visited, int distance, int v) {
		
		visited[v] = true;
		
		if (distance > maxDistance) {
			maxDistance = distance;
			maxIndex = v;
		}
		
		for (Node node : tree[v]) {
			if (visited[node.v]) continue;
			solve(visited, distance + node.w, node.v);
		}
	}
```

## 최종 코드 (Java)

```java
import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.util.ArrayList;
import java.util.StringTokenizer;

public class _1167 {
	
	static int n;
	static ArrayList<Node>[] tree;
	
	static int maxDistance;
	static int maxIndex;

	public static void main(String[] args) throws Exception {
		
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		n = Integer.parseInt(br.readLine());
		tree = new ArrayList[n + 1];
		
		for (int i = 1; i < n + 1; i++) {
			tree[i] = new ArrayList();
		}
		
		for (int i = 0; i < n; i++) {
			StringTokenizer st = new StringTokenizer(br.readLine());
			int v = Integer.parseInt(st.nextToken());
			while (true) {
				int nextV = Integer.parseInt(st.nextToken());
				if (nextV == -1) break;
				int nextD = Integer.parseInt(st.nextToken());
				tree[v].add(new Node(nextV, nextD));
			}
		}
		
		maxDistance = 0;
		solve(new boolean[n + 1], 0, 1);
		solve(new boolean[n + 1], 0, maxIndex);
		System.out.println(maxDistance);
	}
	
	static void solve(boolean[] visited, int distance, int v) {
		
		visited[v] = true;
		
		if (distance > maxDistance) {
			maxDistance = distance;
			maxIndex = v;
		}
		
		for (Node node : tree[v]) {
			if (visited[node.v]) continue;
			solve(visited, distance + node.w, node.v);
		}
		
	}

}

class Node {
	int v;
	int w;
	
	Node(int v, int w) {
		this.v = v;
		this.w = w;
	}
}

```
