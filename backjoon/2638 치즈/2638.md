## 문제 풀이 방식


위 문제에서 가장 생각하기 어려웠던 부분은 내부 공기와 외부 공기를 어떻게 구분할 것인지이다. 또한, 치즈가 한 시간 단위로 동시에 녹아야 한다는 점을 주의해야한다. 배열을 순회하면서 녹을 예정인 치즈를 탐색 도중에 바로 외부 공기로 바꿔버리면, 아직 탐색하지 않은 인접한 치즈가 방금 바뀐 치즈로 인해 같은 턴에 연달아 녹아버릴 수 있다. 따라서 녹을 예정인 치즈는 상태 값을 다르게 두거나, 자료구조를 통해 따로 저장해두어야 한다.

### 1. BFS를 통해 외부 공기를 체크 및 초기화

치즈 내부에 갇힌 공기와 외부 공기를 구분하기 위해, 항상 외부 공기임이 보장되는 `(0, 0)`에서부터 BFS 탐색을 시작한다. 

- 외부 공기는 `-1`, 치즈는 `1`, 다음 턴에 녹을 치즈는 `2`, 내부 공기는 `0`으로 상태를 구분했다.
- BFS 탐색 시 치즈(`1`)가 아닌 모든 칸을 방문하며 `-1`로 덮어씌운다.
- 이 과정에서 이전 턴에 녹을 예정이었던 치즈와 내부 공기가 자연스럽게 외부 공기로 전환된다.

```java
static int bfs() {
		int count = 0;
		
		boolean[][] visited = new boolean[n][m];
		Queue<Pair> queue = new ArrayDeque();
		queue.add(new Pair(0, 0));
		visited[0][0] = true;
		
		while (!queue.isEmpty()) {
			Pair pair = queue.poll();
			int x = pair.x;
			int y = pair.y;
			arr[x][y] = -1;
			count++;
			
			for (int i = 0; i < 4; i++) {
				int nx = x + dx[i];
				int ny = y + dy[i];
				
				if (nx < 0 || nx >= n || ny < 0 || ny >= m) continue;
				if (visited[nx][ny]) continue;
				if (arr[nx][ny] == 1) continue;
				visited[nx][ny] = true;
				queue.add(new Pair(nx, ny));
			}
		}
		return count;
	}

```

### 2. 모든 치즈 칸을 검사하며 바뀔 예정인 치즈 칸을 검사

```java
static boolean check(int x, int y) {
		int count = 0;
		for (int i = 0; i < 4; i++) {
			int nx = x + dx[i];
			int ny = y + dy[i];
			
			if (nx < 0 || nx >= n || ny < 0 || ny >= m) continue;
			if (arr[nx][ny] == -1) count++;
		}
		return count >= 2;
	}
```

2차원 배열을 순회하며 치즈(`1`)인 칸만 검사한다. 상하좌우 4방향을 탐색하여 외부 공기(`-1`)와 2면 이상 맞닿아 있다면, 당장 공기로 바꾸지 않고 녹을 예정이라는 의미로 상태를 `2`로 변경한다. 

### 3. 종료 조건

1번(BFS)과 2번(치즈 마킹) 과정을 하나의 사이클로 묶어 반복하며 시간을 증가시킨다. 

이때, `bfs()`가 반환하는 값은 탐색한 외부 공기의 총 개수이다. 만약 이 값이 격자의 전체 크기와 같아진다면, 모든 치즈가 녹아 맵 전체가 외부 공기가 되었다는 뜻이므로 반복을 종료하고 흐른 시간을 반환한다.

```java
static int solve() {
		int time = 0;
		
		while (bfs() != n * m) {
			for (int x = 0; x < n; x++) {
				for (int y = 0; y < m; y++) {
					if (arr[x][y] != 1) continue;
					if (check(x, y)) arr[x][y] = 2;
				}
			}
			time++;
		}
		return time;
	}
```

## 최종 코드 (Java)

```java
import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.util.ArrayDeque;
import java.util.Queue;
import java.util.StringTokenizer;

public class _2638 {
	
	static int[] dx = {1, 0, -1, 0};
	static int[] dy = {0, 1, 0, -1};
	
	static int n;
	static int m;
	static int[][] arr;

	public static void main(String[] args) throws Exception {
		
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = new StringTokenizer(br.readLine());
		
		n = Integer.parseInt(st.nextToken());
		m = Integer.parseInt(st.nextToken());
		arr = new int[n][m];
		
		for (int x = 0; x < n; x++) {
			st = new StringTokenizer(br.readLine());
			for (int y = 0; y < m; y++) {
				arr[x][y] = Integer.parseInt(st.nextToken());
			}
		}
		System.out.println(solve());
	}
	
	static int solve() {
		int time = 0;
		
		while (bfs() != n * m) {
			for (int x = 0; x < n; x++) {
				for (int y = 0; y < m; y++) {
					if (arr[x][y] != 1) continue;
					if (check(x, y)) arr[x][y] = 2;
				}
			}
			time++;
		}
		return time;
	}
	
	static boolean check(int x, int y) {
		int count = 0;
		for (int i = 0; i < 4; i++) {
			int nx = x + dx[i];
			int ny = y + dy[i];
			
			if (nx < 0 || nx >= n || ny < 0 || ny >= m) continue;
			if (arr[nx][ny] == -1) count++;
		}
		return count >= 2;
	}
	
	static int bfs() {
		int count = 0;
		
		boolean[][] visited = new boolean[n][m];
		Queue<Pair> queue = new ArrayDeque();
		queue.add(new Pair(0, 0));
		visited[0][0] = true;
		
		while (!queue.isEmpty()) {
			Pair pair = queue.poll();
			int x = pair.x;
			int y = pair.y;
			arr[x][y] = -1;
			count++;
			
			for (int i = 0; i < 4; i++) {
				int nx = x + dx[i];
				int ny = y + dy[i];
				
				if (nx < 0 || nx >= n || ny < 0 || ny >= m) continue;
				if (visited[nx][ny]) continue;
				if (arr[nx][ny] == 1) continue;
				visited[nx][ny] = true;
				queue.add(new Pair(nx, ny));
			}
		}
		return count;
	}

}

class Pair {
	int x;
	int y;
	
	Pair(int x, int y) {
		this.x = x;
		this.y = y;
	}
}
```
