## 위상 정렬**(Topological Sorting)**


위상 정렬은 방향 그래프를 선형으로 정렬하는 것이다. 위상 정렬을 가장 잘 설명하는 예시로 교육과정의 선수과목 구조를 예로 들 수 있다. 

위상 정렬 알고리즘을 실행한 후, `inDegree` 배열이 꽉 채워지지 않았다면 그래프 내에 사이클이 존재하는 것이다. 이를 통해 사이클을 판별할 수 있다. 

알고리즘을 이해하기 위해서 진입 차수와 진출 차수에 대한 의미를 알아야한다. 

- 진입 차수 : 특정한 노드로 들어오는 간선의 개수
- 진출 차수 : 특정한 노드에서 나가는 간선의 개수

진입차수 배열을 순회하면서 진입 차수가 0인 노드를 큐에 넣는다. 

### 3. 진입차수가 0인 노드의 간선 자르기

![](https://velog.velcdn.com/images/hariaus/post/09f19c24-1ada-4df5-bc14-0b4094df8911/image.png)


노드 1을 탐색하면 이어져 있는 노드가 2와 3이니, 2와 3의 진입차수를 각각 -1 한다. 이 과정을 거치면 진입 차수가 0인 노드가 2와 3으로 이 노드를 큐에 넣는다. 

큐에 있는 노드를 탐색하고, 진입 차수가 0인 노드를 큐에 추가하는 과정을 반복하여 순서를 알 수 있다. 

## 문제 풀이 방식


문제 풀이의 경우, 위상 정렬을 그대로 옮긴 코드이다. 내 코드는 O($V^2$)으로 동작하는데, 진입 차수가 0인 모든 노드를 큐에 한 번에 넣으면 시간 복잡도를 O($V + E$)로 줄일 수 있다. 

## 최종 코드 (Java)


```java
import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.util.ArrayDeque;
import java.util.ArrayList;
import java.util.Deque;
import java.util.StringTokenizer;

public class _2252 {
	
	static int n;
	static int m;
	
	static ArrayList<Integer>[] arr;
	static int[] indegree;
	static Deque<Integer> queue = new ArrayDeque();
	
	static int[] ans;

	public static void main(String[] args) throws Exception {
		
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringBuilder sb = new StringBuilder();
		
		StringTokenizer st = new StringTokenizer(br.readLine());
		n = Integer.parseInt(st.nextToken());
		m = Integer.parseInt(st.nextToken());
		
		arr = new ArrayList[n + 1];
		indegree = new int[n + 1];
		ans = new int[n];

		for (int i = 0; i < n; i++) {
			arr[i + 1] = new ArrayList<>();
		}
		
		for (int i = 0; i < m; i++) {
			st = new StringTokenizer(br.readLine());
			
			int A = Integer.parseInt(st.nextToken());
			int B = Integer.parseInt(st.nextToken());
			arr[A].add(B);
			indegree[B] += 1;
		}
		
		solve();
		for (int i = 0; i < n; i++) {
			sb.append(ans[i]).append(" ");
		}
		System.out.println(sb);
	}
	
	static void solve() {
		
		int idx = 0;
		while (idx < n) {
			for (int i = 1; i < n + 1; i++) {
				if (indegree[i] == 0) {
					indegree[i] = -1;
					ans[idx++] = i;
					queue.add(i);
				}
			}
			
			int node = queue.pollFirst();
			for (int next_node : arr[node]) {
				if (indegree[next_node] <= 0) continue;
				indegree[next_node] -= 1;
			}
		}
	
	}

}

```
