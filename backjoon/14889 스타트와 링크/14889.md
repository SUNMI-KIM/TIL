## 백트래킹


백트래킹은 스택 혹은 재귀로 구현하는 브루트포스 탐색이라고 생각하면 이해하기 쉽다.

## 문제 풀이 방식


처음엔 깊게 생각하지 않고, 모든 능력치를 한 번에 더한 뒤 팀의 값을 빼는 방식으로 해결하려 했다. 하지만 팀 구성이 달라질 때마다 능력치의 합이 달라지므로 이 방식은 잘못된 접근이었다. 

문제의 핵심은 N명은 두 팀으로 나누고, 각 팀의 능력치 차이의 최솟값을 구하는 것이다. 

### 팀을 어떻게 나눌 것 인가?

한 팀을 뽑는 경우 다른 팀은 자동으로 정해진다. 

처음에는 한 팀을 별도의 배열로 뽑는 방식을 생각했지만, 선택 이후 다른 팀을 다시 구성해야 하는 추가 처리가 필요해 이 방법은 사용하지 않았다. 대신 아래와 같은 방식을 선택했다. 

1. boolean 배열을 사용해 팀을 구분한다. 
2. true인 사람은 스타트 팀, false인 사람은 링크 팀으로 나눈다. 
3. 한 팀만 백트래킹으로 선택하면, 나머지 팀은 자동으로 결정된다.

이 방식은 팀이 바로 나뉘고, 배열 하나로 상태를 관리할 수 있다는 장점이 있다. 처음에는 이 방식을 떠올리지 못해, GPT의 도움을 받았다. 

### 각 팀의 능력치 합의 차 구하기

각 팀의 능력치는 이중 반복문을 통해 계산했다. 두 사람이 같은 팀에 속해 있다면 해당 능력치를 합에 더하는 방식이다.

이중 반복문을 사용하면 `(i, r)`와 `(r, i)`를 두 번 더하기 때문에 최종 결과에서 2로 나누었다.

이 부분 역시 백트래킹으로 계산이 가능하지만, 능력치 계산의 최대 시간 복잡도는 `O(N²)`이고 문제 조건이 N ≤ 20 이므로 시간 제한 내에서 충분히 가능하다고 판단했다. 

다음부터는 가능한 한 단순한 방식부터 떠올리고, 필요할 때 더 복잡한 방법을 고민해야겠다고 느꼈다. 

## 최종 코드 (Java)

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.StringTokenizer;

public class _14889 {
	static int min = Integer.MAX_VALUE;

	public static void main(String[] args) throws IOException {
		
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		
		int n = Integer.parseInt(br.readLine());
		int[][] arr = new int[n][n];
		boolean[] team = new boolean[n];
		
		for (int i = 0; i < n; i++) {
			StringTokenizer st = new StringTokenizer(br.readLine());
			for (int r = 0; r < n; r++) {
				int S = Integer.parseInt(st.nextToken());
				arr[i][r] = S;
			}
		}
		
		solve(arr, team, n, 0, 0);
		
		System.out.println(min);
	}
	
	public static void solve(int[][] arr, boolean[] team, int n, int count, int idx) {
		if (count == n / 2) {
			min = Math.min(sum(team, n, arr), min);
			return;
		}
		
		for (int i = idx; i < n; i++) {
			team[i] = true;
			solve(arr, team, n, count + 1, i + 1);
			team[i] = false;
		}
	}
	
	public static int sum(boolean[] team, int n, int[][] arr) {
		int start = 0;
		int link = 0;
		
		for (int i = 0; i < n; i++) {
			for (int r = 0; r < n; r++) {
				if (team[i] == false && team[r] == false) start += arr[i][r] + arr[r][i];
				if (team[i] == true && team[r] == true) link += arr[i][r] + arr[r][i];
			}
		}
		return Math.abs(start - link) / 2;
	}
}
```
